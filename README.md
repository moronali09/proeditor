# Editor Rank Node.js App

Run locally:

```bash
npm install
npm start
```

Admin:
- username: fatema

Features:
- user post create/edit/delete
- video upload
- thumbnail upload OR auto thumbnail from video
- member leaderboard and rank table for সবাই
- admin can manage members and posts

Note:
- This version uses local JSON storage.
- On Vercel, local file storage is not persistent, so a real database is needed for permanent data.
